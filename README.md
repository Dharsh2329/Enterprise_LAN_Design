# Enterprise LAN Design using VLANs, Inter-VLAN Routing & Wireless Integration

## Overview

Designed and implemented a small enterprise network in Cisco Packet Tracer to simulate a real-world multi-department office environment. 
The project focused on network segmentation, scalability, security, and seamless communication between wired and wireless devices.

## Key Features

* Created VLAN 10 (HR Department)
* Created VLAN 20 (IT Department)
* Configured trunk links between switches
* Implemented Router-on-a-Stick for Inter-VLAN Routing
* Configured centralized DHCP for automatic IP address allocation
* Integrated wireless connectivity using an Access Point
* Enabled communication between wired and wireless devices across VLANs

## Technologies Used

* Cisco Packet Tracer
* VLANs
* Trunking
* Inter-VLAN Routing
* DHCP
* Wireless Networking

## Learning Outcomes

* Network segmentation and traffic isolation
* VLAN configuration and management
* Router-on-a-Stick implementation
* Dynamic IP address management using DHCP
* Enterprise LAN design concepts

## Verification

Successfully verified end-to-end connectivity using ping tests between devices in different VLANs and wireless networks.
